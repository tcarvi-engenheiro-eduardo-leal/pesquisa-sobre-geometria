# Versor

- Um vetor com magnitude igual a 1 unidade de medida.
- Exemplos:
    - Versor $ \vec{i} = (1, 0, 0) $
    - Versor $ \vec{j} = (0, 1, 0) $
    - Versor $ \vec{k} = (0, 0, 1) $
- Versor gerado a partir de um vetor qualquer $ \vec{AB} $:
    - Versor = $ \frac{ \vec{AB}}{|\vec{AB}|} $
    - Versor = $ ( \frac{X_B - X_A}{|\vec{AB}|}, \frac{Y_B - Y_A}{|\vec{AB}|}) $
