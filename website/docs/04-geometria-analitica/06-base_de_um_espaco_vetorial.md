# Base de um Espaço Vetorial

## Base Ortonormal de um Vetor
- Uma base é considerada ortonormal se:
    1. Seus vetores são ortogonais ( $ \vec{v_1} \perp \vec{v_1} $ )
    2. Seus vetores são unitários: ( $ \| \vec{v_1} \| = \| \vec{v_1} \| = 1 $ )

## Base canônica em R2
- $ \vec{i} = (1,0) $
- $ \vec{j} = (0,1) $

## Base canônica em R3
- $ \vec{i} = (1,0,0) $
- $ \vec{j} = (0,1,0) $
- $ \vec{k} = (0,0,1) $
